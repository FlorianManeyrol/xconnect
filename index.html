X Connect

Garanteeing user identity by relying on their account at third party providers

What's X Connect?

X Connect is a service that avoid the hassle of handling connection credentials for each and every web application where a user has to connect.

X Connect allows a user to open a session at a web application by using their credentials at a third party provider, that authenticates the user on behalf of the web application.
